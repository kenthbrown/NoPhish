# System Overview

## Purpose

NoPhish is intended to demonstrate how a web application could help users review suspicious URLs or email content and surface phishing risk indicators.

## High-Level Components

- A frontend where users can paste a URL or email text
- A backend service that evaluates the submitted content
- Documentation that explains the architecture and future implementation plan

## Current Status

This project is in the setup phase. Only the folder structure and starter documentation have been created so far.
